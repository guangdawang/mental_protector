## 🚀 快速开始

### 首次使用

1. **打开应用** → 自动显示引导页面（6页）
2. **浏览引导** → 了解各项功能
3. **点击"开始使用"** → 进入主页

### 记录情绪

**方法一：手动记录**
1. 点击"记录情绪"按钮
2. 选择最符合当前心情的选项
3. （可选）添加备注
4. 点击"保存记录"

**方法二：自动记录**
- 检测到危机内容时自动记录
- 情绪状态自动更新

### 查看历史

1. 点击右上角"历史记录"图标
2. 查看按日期分组的情绪变化
3. 查看趋势图表了解情绪走向

### 进入安全岛

**方法一：自动进入**
- 检测到危机内容时自动进入

**方法二：手动进入**
- 点击"手动进入安全岛"按钮

## 📚 功能详解

### 1. 危机检测

#### 工作原理
- **关键词检测**：匹配13个危机关键词
- **输入模式分析**：检测反复删除、极速输入、深夜使用
- **智能评分**：综合计算危机程度（0-10分）

#### 触发条件
```
高危（8-10分）：
- "自杀"、"不想活了"、"死了算了"等
→ 立即进入安全岛

中危（6-7分）：
- "跳楼"、"割腕"、"吃药"等
→ 提示进入安全岛

低危（4-5分）：
- "解脱"、"活不下去"等
→ 静默记录

正常（0-3分）：
- 无关键词或模式异常
→ 正常使用
```

### 2. 安全岛

#### 功能说明
```
┌─────────────────────────────────┐
│    顶部：检测提示信息          │
│                             │
│    中间：呼吸圆圈            │
│    - 吸气（2秒）             │
│    - 屏息（1秒）             │
│    - 呼气（3秒）             │
│    - 6秒循环                 │
│                             │
│    底部：求助按钮            │
│    - 一键拨打热线            │
│    - 联系紧急联系人          │
│    - 返回按钮                │
└─────────────────────────────────┘
```

#### 呼吸练习
1. **跟随圆圈**：圆圈膨胀时吸气，收缩时呼气
2. **保持专注**：注视圆圈，深呼吸
3. **放松身体**：配合呼吸，放松肩膀和眼睛
4. **持续练习**：建议练习3-5分钟

### 3. 紧急联系人

#### 设置方法
1. 进入设置页面
2. 点击"添加紧急联系人"
3. 填写姓名、电话、关系
4. 最多可添加3个

#### 倒计时机制
```
检测到危机 + 已启用功能
  ↓
自动进入安全岛
  ↓
用户点击"联系紧急联系人"
  ↓
显示10秒倒计时对话框
  ├─ 5秒：思考时间
  ├─ 8秒：轻微振动提醒
  └─ 10秒：自动发送短信
  ↓
发送成功（用户可随时取消）
```

#### 3个月过期机制
- 关闭紧急联系功能后，90天后自动过期
- 过期后需重新开启
- 系统会在关闭60天后提醒

### 4. 数据备份

#### 备份流程
```
设置 → 备份数据
  ↓
自动加密（AES-256）
  ↓
生成.heart文件
  ↓
保存到应用目录
  ↓
可选择分享到：
- 邮箱
- 云盘
- USB传输
```

#### 恢复流程
```
设置 → 恢复数据
  ↓
选择.heart备份文件
  ↓
输入密码（如设置了）
  ↓
自动解密并导入
  ↓
重启应用生效
```

## ❓ 常见问题

### Q: 危机检测准确吗？

**A:** 检测基于关键词和输入模式，可能有误判。如果误判：
- 点击"返回"退出安全岛
- 检测不影响正常使用

### Q: 如何关闭危机检测？

**A:** 目前无法完全关闭，但可以：
- 避免输入关键词
- 在正常时间使用（避免深夜）
- 正常输入速度和模式

### Q: 数据备份安全吗？

**A:** 是的，备份文件使用：
- AES-256加密（军用级别）
- SHA-256完整性验证
- 完全离线，不上传云端

### Q: 紧急联系人会立即发送吗？

**A:** 不会，需要：
1. 用户预先设置并开启功能
2. 用户主动点击"联系紧急联系人"
3. 10秒倒计时结束（可随时取消）

### Q: 如何重置所有数据？

**A:**
1. 进入设置页面
2. 滚动到"数据管理"
3. 点击"清除所有数据"
4. 确认后清除（不可撤销）

### Q: 情绪记录会自动同步到其他设备吗？

**A:** 不会自动同步。如需同步：
1. 备份数据（导出.heart文件）
2. 通过邮件/云盘发送到新设备
3. 在新设备上恢复数据

## 🔧 高级功能

### 搜索情绪记录

在历史记录页面：
1. 点击搜索图标
2. 输入关键词：
   - 日期：如"12月"、"今天"
   - 分数：如"5分"
   - 备注：如"开心"、"难过"
3. 查看过滤后的结果

### 情绪趋势图

查看图表了解情绪走向：
- **上升曲线**：情绪在改善
- **下降曲线**：需要注意
- **波动较大**：情绪不稳定

### 月度提醒

系统会在每月1号提醒：
- 检查紧急联系人设置
- 确认紧急联系功能状态

### 音频和振动设置

在设置中可调节：
- **音频播放**：开关安全岛心跳声
- **音频音量**：0-100%调节
- **振动反馈**：开关呼吸和操作振动
